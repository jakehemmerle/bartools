# bartools

A small collection of utilities for people who spend too much time behind (or thinking about) a bar.

## What's in here

- **pour-timer** — a tiny CLI that counts free-pour seconds so you can calibrate against a jigger
- **spec-book** — plain-text cocktail specs, versioned so you can actually track what changed between "the Tuesday Negroni" and "the good Negroni"
- **stock-check** — a quick inventory pass that flags anything under par before your next order

## Why

Because sticky notes behind the well are not a backup strategy, and because "I'll remember" is a lie every bartender tells themselves around 11pm on a Friday.

## Status

Early and opinionated. Interfaces will change. Bring your own shaker.

## Install

Clone it, poke around, complain about the defaults, send a PR.

```sh
git clone https://github.com/<you>/bartools
cd bartools
```
