# Golden Set Approved

This folder contains the approved BARBACK web dashboard design references collected into one repo-local bundle.

Use this folder when:
- sharing the approved screen set with another team
- re-uploading images into Stitch or another design tool
- checking implementation against the chosen visual references

## Contents

Each approved screen is stored as a copied source folder containing:
- `code.html`
- `screen.png`

There is also an `images/` folder with friendly, upload-ready filenames.
